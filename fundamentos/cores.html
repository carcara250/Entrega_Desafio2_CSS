Cores

Existem varias formas de definimos cores utilizada no CSS.

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hex-transparencia
  • rgb
  • rgba
  • hsl
  • hsla

HSL


hsl (hue, saturation, lightness)

    Hue (matiz): é o grau na roda de cores (de 0 a 360 graus)
  • 0 ou 360® é vermelho
  • 120 é verde
  • 240 é azul
    Saturation (saturação): valor percentual
  • 0% Um tom de cinza
  • 100% cor total
    lightness (luminosidade):luminosidade da cor
  • 0% é preto
  • 100% é branco
